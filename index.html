Welcome to Paperbacks & Pitstops

Hi, I'm Kezia. This website was developed as part of my ICT171 Cloud Server Project and serves as my personal space to share my interests in books, music, poetry and Formula 1.

The website is hosted on Microsoft Azure, secured using HTTPS through Let's Encrypt and managed through GitHub version control.

☁️ About This Project

Paperbacks & Pitstops is a cloud-hosted personal blog developed for ICT171 Introduction to Server Environments and Architectures. The website is hosted on a Microsoft Azure virtual machine, connected to a custom domain name and secured using SSL/TLS certificates from Let's Encrypt.
